# 🚀 Быстрый старт веб-панели

## За 3 минуты до запуска

### 1. Установите зависимости
```bash
npm install
```

### 2. Настройте `.env`
```env
# Основные настройки бота
BOT_TOKEN=ваш_токен_от_botfather
TIMEZONE=Europe/Moscow

# Веб-панель
ENABLE_DASHBOARD=true
DASHBOARD_PORT=3000
DASHBOARD_PASSWORD=ваш_секретный_пароль

# Администраторы
ADMIN_IDS=ваш_telegram_id
```

### 3. Запустите
```bash
npm start
```

### 4. Откройте в браузере
```
http://localhost:3000
```

## ✅ Готово!

Введите пароль из `DASHBOARD_PASSWORD` и получите доступ к статистике.

---

## 📊 Что доступно в панели?

### Главная страница (`/`)
- **Общая статистика:** всего пользователей, активные сегодня/неделя
- **Топ-5 пользователей** по текущим сериям
- **Статистика вопросов:** всего, неиспользованных, предложений
- **Автообновление** каждые 30 секунд

### Справка по командам (`/commands`)
- Полный список всех админских команд
- Примеры использования
- Быстрые сценарии
- Цветовая индикация опасности команд

---

## 🔐 Безопасность

### ⚠️ Важно!

1. **Смените пароль по умолчанию:**
   ```env
   DASHBOARD_PASSWORD=My$ecureP@ssw0rd!2024
   ```

2. **Не публикуйте `.env`** в git (уже в `.gitignore`)

3. **В продакшене используйте HTTPS:**
   - Настройте nginx с SSL
   - Используйте Let's Encrypt

## 🔧 Решение проблем

### Панель не запускается?

**Ошибка:** `Cannot find module 'express'`

```bash
npm install express
```

---

**Ошибка:** `Port 3000 is already in use`

Измените порт в `.env`:
```env
DASHBOARD_PORT=8080
```

---

### Не могу войти?

1. Проверьте пароль в `.env`
2. Перезапустите бота: `npm start`
3. Очистите кэш браузера (Cmd+Shift+R)

---

### Нет данных?

Убедитесь что:
- База данных создана (`database.db` существует)
- Есть хотя бы один пользователь
- Бот запущен и работает
